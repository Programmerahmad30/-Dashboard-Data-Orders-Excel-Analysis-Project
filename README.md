📊 Dashboard Data Orders – Excel Analysis Project
📌 Overview

This project presents a comprehensive Excel dashboard for analyzing food order data.
The goal is to transform raw data into meaningful insights that help understand sales performance, customer behavior, and business trends.

🎯 Project Objectives
Analyze overall business performance using key KPIs
Track sales trends over time (daily, weekly, monthly)
Understand customer satisfaction through ratings
Identify top-performing cities and regions
Compare sales distribution by food categories (Veg vs Non-Veg)
📈 Key KPIs

The dashboard highlights the following metrics:

Total Sales → Overall revenue generated
Average Rating → Customer satisfaction level
Total Orders → Number of orders placed
Ratings Count → Total number of reviews
Average Order Value (AOV) → Revenue per order
📊 Dashboard Features

The report includes multiple visualizations:

📅 Time-Based Analysis
Monthly Sales Trend → Tracks revenue changes across months
Daily Sales Trend → Shows performance by day of the week
Weekly Trend Analysis → Identifies consistency and peak periods
🍽️ Category Insights
Sales by Food Type (Veg vs Non-Veg)
→ Compares contribution of each category
🗺️ Geographic Analysis
Sales by State (Map Visualization)
→ Displays regional performance
🏆 Performance Insights
Top 5 Cities by Sales → Highlights best-performing locations
Quarterly Summary → Combines:
Sales
Ratings
Orders
